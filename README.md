# 普瑞賽斯Bot

一個以明日方舟為主題的 Discord 機器人，提供幹員資料查詢、遊戲資訊、娛樂功能等。

## 功能

### 幹員查詢
| 指令 | 說明 |
|------|------|
| `/幹員資料` | 基本屬性、職業、立繪、製作人員等 |
| `/幹員檔案` | 基礎檔案、體檢、履歷、診斷等背景故事 |
| `/幹員技能` | 技能（Lv.7 / 專精）、天賦、後勤技能、模組 |
| `/幹員時裝` | 時裝圖片、品牌、畫師與取得方式 |
| `/幹員素材計算` | 技能專精與模組解鎖所需材料 |

### 遊戲資訊
| 指令 | 說明 |
|------|------|
| `/集成戰略資訊` | 難度資料、分隊介紹或藏品資訊（支援自動完成） |
| `/陸服卡池未來視` | 陸服限時尋訪活動一覽（由新至舊） |
| `/地區資料` | 泰拉大陸地區簡介、國徽與場景圖 |

### 角色資訊
| 指令 | 說明 |
|------|------|
| `/角色真名` | 查詢幹員或劇情角色的真實姓名與出處 |
| `/劇情角色` | 劇情角色的簡介、出處與立繪 |

### 娛樂功能
| 指令 | 說明 |
|------|------|
| `/抽角色 [偏好]` | 從幹員中隨機抽取老公或老婆 |
| `/抽角色擴充版 [偏好]` | 幹員＋劇情角色全員參與抽籤 |
| `/語音猜角色 [模式]` | 聆聽幹員 JP 語音猜幹員名稱，支援全語音與 Arknights 模式 |

| 指令 | 說明 |
|------|------|
| `/幫助` | 顯示所有指令說明 |

## 技術

- **語言**：Python 3.13
- **函式庫**：[discord.py](https://github.com/Rapptz/discord.py) 2.x
- **資料來源**：[PRTS Wiki](https://prts.wiki)、[Arknights Wiki.gg](https://arknights.wiki.gg)
- **部署**：[Railway](https://railway.com)

## 部署

### 本地執行
```bash
pip install -r requirements.txt
python bot.py
```

### Railway 部署
```bash
railway up
```
